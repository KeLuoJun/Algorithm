# 算法手册 - Python实现

## 📖 项目概述

本项目是一个全面且易于理解的算法手册，采用Python语言实现，涵盖了计算机科学中的经典算法和数据结构。每个算法都配有详细的解释说明、标准化代码实现和丰富的注释，旨在帮助初学者理解算法原理并能够举一反三。

## 🎯 目标用户

- 算法初学者：理解基础算法概念
- 编程爱好者：掌握算法实现方法  
- 面试准备者：通过经典案例学习应用场景
- 开发者：快速查找所需算法内容

## 📚 算法分类详解

### 🔍 搜索算法
- **BFS (广度优先搜索)**: 层次遍历，最短路径
- **DFS (深度优先搜索)**: 深度遍历，回溯算法
- **双向广搜**: 优化的BFS，减少搜索空间
- **二分查找**: 有序数组的高效查找

### 💡 动态规划 (DP)
- **一维DP**: 线性动态规划问题
- **二维DP**: 二维状态转移
- **区间DP**: 区间上的动态规划
- **状压DP**: 状态压缩动态规划
- **背包问题**: 01背包、完全背包、多重背包等

### 📊 数据结构
- **单调栈**: 维护单调性的栈结构
- **单调队列**: 滑动窗口最值问题
- **树状数组**: 高效的区间查询和更新
- **并查集**: 动态连通性问题
- **哈希表**: 快速查找和去重

### 🎯 图算法
- **Dijkstra**: 单源最短路径
- **最小生成树**: Kruskal和Prim算法
- **拓扑排序**: 有向无环图的线性排序

### ⚡ 优化技巧
- **双指针**: 减少时间复杂度的技巧
- **滑动窗口**: 子数组/子串问题的优化
- **前缀和**: 快速计算区间和
- **差分数组**: 区间更新的优化

### 🎲 其他算法
- **排序算法**: 快排、归并、堆排序等
- **贪心算法**: 局部最优解策略
- **博弈论**: 游戏理论相关算法
