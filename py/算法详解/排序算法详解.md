# 排序算法详解

## 📖 概述

排序算法是计算机科学中最基础也是最重要的算法之一。本章节详细介绍了几种经典的排序算法，包括它们的原理、实现和应用场景。

---

## 🚀 快速排序 (Quick Sort)

### 算法原理

快速排序是一种基于分治思想的排序算法，通过选择一个基准元素，将数组分为两部分：小于基准的元素和大于基准的元素，然后递归地对这两部分进行排序。

### 核心思想
1. **选择基准**: 从数组中选择一个元素作为基准(pivot)
2. **分区操作**: 重新排列数组，使得小于基准的元素在左边，大于基准的元素在右边
3. **递归排序**: 对左右两个子数组递归地进行快速排序

### 算法特点
- **时间复杂度**: 
  - 最好情况: O(n log n) - 每次都能均匀分割
  - 平均情况: O(n log n)
  - 最坏情况: O(n²) - 每次选择的基准都是最大或最小值
- **空间复杂度**: O(log n) - 递归调用栈的深度
- **稳定性**: 不稳定排序
- **原地排序**: 是

### 代码实现与详解

```python
# 随机快排 - 优化版本
import random

class QuickSort:
    def __init__(self, arr):
        self.arr = arr  
        self.n = len(arr)  
        self.first = None  # 小于基准值x的越界左边界
        self.last = None   # 大于基准值x的越界右边界
    
    def _swap(self, i, j):
        """交换数组中两个位置的元素"""
        self.arr[i], self.arr[j] = self.arr[j], self.arr[i]

    def _partition(self, l, r, x):
        """
        三路快排的分区函数
        将数组[l, r]分区，使得：
        - [l, first-1]: 小于x的元素
        - [first, last]: 等于x的元素  
        - [last+1, r]: 大于x的元素
        """
        self.first = l  # 等于x区域的左边界
        self.last = r   # 等于x区域的右边界
        i = l           # 当前处理的位置
        
        while i <= self.last:
            if self.arr[i] == x:
                i += 1  # 如果当前元素等于x，继续下一个元素
            elif self.arr[i] < x:
                # 当前元素小于x，放到小于区域
                self._swap(i, self.first)  
                self.first += 1  # 扩大小于区域
                i += 1
            else:
                # 当前元素大于x，放到大于区域
                self._swap(i, self.last)  
                self.last -= 1   # 扩大大于区域
                # 注意：这里i不自增，因为交换过来的元素还没处理

    def _quick_sort(self, l, r):
        """递归实现快速排序"""
        if l >= r:
            return  # 基础情况：数组长度<=1，无需排序
        
        # 随机选择基准元素，避免最坏情况
        x = self.arr[random.randint(l, r)] 
        
        # 进行三路分区
        self._partition(l, r, x)  
        
        # 计算新的子数组边界
        left = self.first - 1   # 小于区域的右边界
        right = self.last + 1   # 大于区域的左边界
        
        # 递归排序左右两部分
        self._quick_sort(l, left)     # 排序小于区域
        self._quick_sort(right, r)    # 排序大于区域
        # 注意：等于区域不需要排序

    def get_sorted_arr(self):
        """获取排序后的数组"""
        self._quick_sort(0, self.n - 1)  
        return self.arr  

def main() -> None:
    arr = list(map(int, input("请输入待排序数组：\n").split()))  
    quick_sort = QuickSort(arr)  
    print("快速排序结果：\n", quick_sort.get_sorted_arr())  

if __name__ == '__main__':
    main()
```

### 算法优化技巧

1. **随机化基准选择**: 避免最坏情况的发生
2. **三路快排**: 处理大量重复元素的情况
3. **小数组优化**: 当数组长度小于某个阈值时，使用插入排序
4. **尾递归优化**: 减少递归调用栈的深度

### 应用场景
- 通用排序算法，适用于大多数情况
- 标准库中的排序实现（如C++的sort）
- 需要原地排序的场景

## 🔄 归并排序 (Merge Sort)

### 算法原理

归并排序是一种稳定的分治排序算法，它将数组分成两半，分别排序，然后将结果合并。

### 核心思想
1. **分解**: 将数组分成两个子数组
2. **解决**: 递归地排序两个子数组
3. **合并**: 将两个已排序的子数组合并成一个有序数组

### 算法特点
- **时间复杂度**: O(n log n) - 所有情况下都是这个复杂度
- **空间复杂度**: O(n) - 需要额外的数组空间
- **稳定性**: 稳定排序
- **原地排序**: 否

### 代码实现与详解

```python
class MergeSort:
    def __init__(self, arr):
        self.arr = arr
        self.n = len(arr)
        self.temp = [0] * self.n  # 辅助数组，用于合并操作
    
    def _merge(self, l, mid, r):
        """
        合并两个已排序的子数组
        [l, mid] 和 [mid+1, r] 都是已排序的
        """
        # 将原数组的元素复制到辅助数组
        for i in range(l, r + 1):
            self.temp[i] = self.arr[i]
        
        i, j = l, mid + 1  # 两个子数组的起始位置
        k = l              # 合并后数组的当前位置
        
        # 比较两个子数组的元素，将较小的放入原数组
        while i <= mid and j <= r:
            if self.temp[i] <= self.temp[j]:
                self.arr[k] = self.temp[i]
                i += 1
            else:
                self.arr[k] = self.temp[j]
                j += 1
            k += 1
        
        # 处理剩余元素
        while i <= mid:
            self.arr[k] = self.temp[i]
            i += 1
            k += 1
        
        while j <= r:
            self.arr[k] = self.temp[j]
            j += 1
            k += 1
    
    def _merge_sort(self, l, r):
        """递归实现归并排序"""
        if l >= r:
            return  # 基础情况：数组长度<=1
        
        mid = (l + r) // 2  # 找到中点
        
        # 递归排序左右两部分
        self._merge_sort(l, mid)
        self._merge_sort(mid + 1, r)
        
        # 合并两个已排序的部分
        self._merge(l, mid, r)
    
    def get_sorted_arr(self):
        """获取排序后的数组"""
        self._merge_sort(0, self.n - 1)
        return self.arr

def main() -> None:
    arr = list(map(int, input("请输入待排序数组：\n").split()))
    merge_sort = MergeSort(arr)
    print("归并排序结果：\n", merge_sort.get_sorted_arr())

if __name__ == '__main__':
    main()
```

### 应用场景
- 需要稳定排序的场景
- 外部排序（处理大文件）
- 链表排序
- 逆序对计算

## 🏔️ 堆排序 (Heap Sort)

### 算法原理

堆排序利用堆这种数据结构来进行排序。它首先将数组构建成一个最大堆，然后不断地将堆顶元素（最大值）与堆的最后一个元素交换，并重新调整堆。

### 核心思想
1. **建堆**: 将无序数组构建成最大堆
2. **排序**: 重复执行"取出堆顶元素，重新调整堆"的操作

### 算法特点
- **时间复杂度**: O(n log n) - 所有情况
- **空间复杂度**: O(1) - 原地排序
- **稳定性**: 不稳定排序
- **原地排序**: 是

### 代码实现与详解

```python
class HeapSort:
    def __init__(self, arr):
        self.arr = arr
        self.n = len(arr)
    
    def _heapify(self, n, i):
        """
        调整以i为根的子树，使其满足最大堆性质
        n: 堆的大小
        i: 当前节点的索引
        """
        largest = i      # 假设当前节点是最大的
        left = 2 * i + 1    # 左子节点
        right = 2 * i + 2   # 右子节点
        
        # 如果左子节点存在且大于当前最大值
        if left < n and self.arr[left] > self.arr[largest]:
            largest = left
        
        # 如果右子节点存在且大于当前最大值
        if right < n and self.arr[right] > self.arr[largest]:
            largest = right
        
        # 如果最大值不是当前节点，则交换并继续调整
        if largest != i:
            self.arr[i], self.arr[largest] = self.arr[largest], self.arr[i]
            self._heapify(n, largest)  # 递归调整受影响的子树
    
    def _build_max_heap(self):
        """构建最大堆"""
        # 从最后一个非叶子节点开始，向上调整
        for i in range(self.n // 2 - 1, -1, -1):
            self._heapify(self.n, i)
    
    def get_sorted_arr(self):
        """堆排序主函数"""
        # 1. 构建最大堆
        self._build_max_heap()
        
        # 2. 逐个取出堆顶元素
        for i in range(self.n - 1, 0, -1):
            # 将堆顶元素（最大值）与最后一个元素交换
            self.arr[0], self.arr[i] = self.arr[i], self.arr[0]
            
            # 重新调整堆（堆的大小减1）
            self._heapify(i, 0)
        
        return self.arr

def main() -> None:
    arr = list(map(int, input("请输入待排序数组：\n").split()))
    heap_sort = HeapSort(arr)
    print("堆排序结果：\n", heap_sort.get_sorted_arr())

if __name__ == '__main__':
    main()
```

### 应用场景
- 需要原地排序且空间有限的场景
- 优先队列的实现
- Top-K问题

## 📝 插入排序 (Insertion Sort)

### 算法原理

插入排序通过构建有序序列，对于未排序数据，在已排序序列中从后向前扫描，找到相应位置并插入。

### 核心思想
1. 从第二个元素开始，将其与前面的元素比较
2. 找到合适的位置插入
3. 重复此过程直到所有元素都被插入到正确位置

### 算法特点
- **时间复杂度**: 
  - 最好情况: O(n) - 数组已经有序
  - 平均情况: O(n²)
  - 最坏情况: O(n²) - 数组逆序
- **空间复杂度**: O(1)
- **稳定性**: 稳定排序
- **原地排序**: 是

### 代码实现与详解

```python
class InsertionSort:
    def __init__(self, arr):
        self.arr = arr
        self.n = len(arr)
    
    def get_sorted_arr(self):
        """插入排序主函数"""
        for i in range(1, self.n):
            key = self.arr[i]  # 当前要插入的元素
            j = i - 1          # 已排序部分的最后一个元素的索引
            
            # 将大于key的元素向后移动
            while j >= 0 and self.arr[j] > key:
                self.arr[j + 1] = self.arr[j]
                j -= 1
            
            # 插入key到正确位置
            self.arr[j + 1] = key
        
        return self.arr

def main() -> None:
    arr = list(map(int, input("请输入待排序数组：\n").split()))
    insertion_sort = InsertionSort(arr)
    print("插入排序结果：\n", insertion_sort.get_sorted_arr())

if __name__ == '__main__':
    main()
```

### 应用场景
- 小规模数据排序
- 数组基本有序的情况
- 作为其他排序算法的优化（如快排的小数组优化）

## 📊 排序算法比较

| 算法 | 时间复杂度(最好) | 时间复杂度(平均) | 时间复杂度(最坏) | 空间复杂度 | 稳定性 | 原地排序 |
|------|-----------------|-----------------|-----------------|-----------|--------|----------|
| 快速排序 | O(n log n) | O(n log n) | O(n²) | O(log n) | 不稳定 | 是 |
| 归并排序 | O(n log n) | O(n log n) | O(n log n) | O(n) | 稳定 | 否 |
| 堆排序 | O(n log n) | O(n log n) | O(n log n) | O(1) | 不稳定 | 是 |
| 插入排序 | O(n) | O(n²) | O(n²) | O(1) | 稳定 | 是 |

## 🎯 选择建议

1. **通用场景**: 快速排序（随机化版本）
2. **需要稳定排序**: 归并排序
3. **空间受限**: 堆排序
4. **小规模数据**: 插入排序
5. **基本有序**: 插入排序
